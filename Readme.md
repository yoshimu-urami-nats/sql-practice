# SQL Practice Repository

このリポジトリは、MySQL を利用した SQL（DDL / DML）学習ログをまとめたものです。  
AWS 環境・A5:SQL Mk-2 を使いながら、Step形式で体系的に学習を進めています。

---

## 📌 開発環境（Environment）

| 項目 | 内容 |
|------|------|
| OS | Windows 11 |
| DB | MySQL 8.0.43-0ubuntu0.24.04.2（AWS EC2） |
| GUI クライアント | A5:SQL Mk-2 |
| エディタ | VSCode（SQL & Markdown） |
| バージョン管理 | GitHub |



---

## 🧰 使用ツール（Tools）

### ◼ A5:SQL Mk-2
- MySQL への GUI 接続  
- DDL / DML の実験  
- テーブル作成 / ER 図 / クエリ実行  
- .sql ファイルの保存

### ◼ Visual Studio Code
- Markdown 学習ログ  
- GitHub への push 運用
